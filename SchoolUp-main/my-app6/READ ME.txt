	Заметка для ориентирования на местности.

	НИ ЗА ЧТО НИКОГДА НЕ ТРОГАТЬ:
	1. файлы типа BAK
	2. index.js ( связывает index.html из папки public c App.js)
	3. reportWebVitals.js
	4. setupTests.js 

	ПОЯСНЯЮ:
		App.js - грубо говоря, простой файл-навигатор для всей остальной системы. 
					Просто так в него лезть не рекомендую.
		
		LeftPanel.js и TopPanel.js - в данный момент в разработке.
										Первый - шаблон панели внутри системы.
										Второй - неизменяемый набор ссылок.
		
		login.js - авторизация. В данный момент с весьма проходным процессом входа, т.е. без юзера.
		
		Profile.js - заготовка для "личного кабинета" пользователя. По умолчанию в login.js задала
						переход на него после входа. Мне кажется это более чем логичным.
		
		ProjectCSS.ssc - общий на весь проект CSS-файл. Насчет навигации внутри него пока что думаю.
		
		users.json - заготовка для пользователей (преподавателей).
